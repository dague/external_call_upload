# external_call_upload

Набор документов и утилит для интеграции загрузки записей звонков.

## Назначение

Здесь всё, что нужно для реализации загрузки: спецификация API, пример кода, инструкции по эксплуатации.

## Структура

**Спецификация API** (рабочие документы для разработки):
- `integration_guide.md` — шаги интеграции, авторизация, ограничения, ретраи
- `openapi.yaml` — схема API (request/response)

**Эксплуатация**:
- `operations_runbook.md` — типовые сбои, диагностика, рекомендации по метрикам

**Пример реализации + утилита для пилота**:
- `external_call_uploader.py` — Python CLI: init → upload по signed URL → finalize. Может служить эталоном при разработке своей интеграции и использоваться для загрузки на этапе пилота, пока собственная реализация не готова
- `external_call_uploader.md` — документация по CLI (аргументы, ретраи, типовые проблемы)

## Быстрый старт (CLI)

Для пилота или проверки:

```bash
pip install requests
python external_call_uploader.py /path/to/call.mp3 \
  --token "<TOKEN>" \
  --operator-phone "+79990001122" \
  --client-phone "+79990003344" \
  --start-time "2026-03-01T12:30:00+03:00" \
  --call-type in
```

Токен выдаётся при подключении интеграции.
